# Miracle Clickz — Cinematic Wedding Stories

An award-grade, pure-black luxury photography experience built with Next.js 15, React 19, TypeScript, Tailwind CSS, GSAP (ScrollTrigger), Lenis smooth scroll and Three.js particles.

## Run it

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # production build
```

## What's inside

- **Landing** — camera-shutter (iris) preloader with loading percentage, opening into the homepage
- **Hero** — slow cinematic zoom backdrop, 85% black gradient, split-character title reveal, magnetic CTAs, scroll indicator; text fades & scene scales on scroll
- **Portfolio** — floating 3D tilt cards with glass reflections, category filters, scroll-depth parallax; opens a fullscreen viewer (keyboard ←/→/Esc, counter, progress bar, slow zoom)
- **Our Story** — growing gold timeline with a camera icon that follows scroll; every line reveals differently (mask, blur, 3D rotate, scale, tracking)
- **Services / Stats / Films / Testimonials / Instagram / Process / Awards / FAQ / Booking / Footer** — each section enters with its own animation flavour
- **Everywhere** — Lenis smooth scroll, glowing ring cursor, gold Three.js dust reacting to the mouse, light rays, film grain, subtle glassmorphism

## Swapping in real media

All imagery is generated by `components/CinemaArt.tsx` (offline cinematic placeholders). Replace its usage with `next/image` / `<video>` elements:

- Hero backdrop → drop a muted, looped `<video>` into `components/Hero.tsx` (the slow-zoom + overlay wrappers already exist)
- Portfolio / Films / Instagram → swap `<CinemaArt …/>` for your photos; data lives in `lib/data.ts`
- Booking form → wire the `submit` handler in `components/BookNow.tsx` to your API route or a service like Formspree

## Notes

- All motion respects `prefers-reduced-motion`
- SplitText-style reveals use a custom free splitter (`lib/gsap.ts`) — no paid GSAP plugins required
- Animation is unified on GSAP + Lenis for consistent 60fps (no competing animation libraries)
- SEO: metadata + Open Graph + JSON-LD (`app/layout.tsx`)
